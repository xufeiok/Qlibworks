## 一、项目信息存储结构

**存放根目录：** `/opt/knowledge/地产营销`

每个项目存为一个独立笔记/文件，格式如下：

```
/opt/knowledge/地产营销/
└── [项目名称]/
    ├── project_info.md    # 项目基本信息
    ├── marketing_plan.md  # 营销方案
    ├── copywriting/       # 文案素材
    ├── competitive/       # 竞品分析
    ├── research/          # 市场调研
    ├── attachments/       # 原始附件
    └── data.duckdb       # 结构化数据文件（可选）
```

**关于 DuckDB 数据文件（重要！）：**

知识库支持在项目目录下放置 `.duckdb` 文件来存储结构化数据。

| 项目 | 说明 |
|------|------|
| 路径示例 | `/opt/knowledge/地产营销/西悦云庭/sales_data.duckdb` |
| 连接方式 | Python `import duckdb; conn = duckdb.connect('/opt/knowledge/地产营销/.../xxx.duckdb')` |
| 依赖 | `pip install duckdb`（已装在 `/opt/data/venv/bin/pip`） |
| 优势 | 分析型查询比 SQLite 快 10-100 倍，支持大数据量（GB~TB），可直接读取 CSV/Parquet |
| 适用场景 | 销售数据统计、客户数据整理、竞品数据对比、批量文案素材标注 |

**DuckDB 使用示例（可直接运行）：**

```python
import duckdb

db_path = '/opt/knowledge/地产营销/西悦云庭/sales_data.duckdb'
conn = duckdb.connect(db_path)
cur = conn.cursor()

# 建表
cur.execute('''
    CREATE TABLE IF NOT EXISTS clients (
        id INTEGER PRIMARY KEY,
        name VARCHAR,
        phone VARCHAR,
        intention_level VARCHAR,  -- 高/中/低
        follow_up_date DATE,
        agent VARCHAR
    )
''')

# 插入
cur.execute("INSERT INTO clients VALUES (1, '张三', '13800138000', '高', '2026-05-01', '李四')")
conn.commit()

# 查询
cur.execute("SELECT * FROM clients WHERE intention_level = '高'")
for row in cur.fetchall():
    print(row)

conn.close()
```

**分析型查询示例：**

```python
# 按意向等级统计客户数量
cur.execute('''
    SELECT intention_level, COUNT(*) as cnt
    FROM clients
    GROUP BY intention_level
    ORDER BY cnt DESC
''')
print(cur.fetchall())
```

## 二、广告文案创作方法论

### 核心理论框架

#### 空手（叶伟）《文案三件套》

**核心主张：文案三件套 = 货架思维 + 戏剧性 + 口语化**

| 概念 | 解释 |
|------|------|
| 货架思维 | 文案放在货架上能被注意到吗？先解决"被人看见"的问题 |
| 戏剧性 | 制造冲突、转折、意外，让文案像故事一样有张力 |
| 口语化 | 说出来顺嘴，听起来耳熟，像人话 |

> "文案不是文学，文案是销售"
> "最好的文案是让消费者觉得这是他自己想出来的"
> "要么不说话，要么说话就是钉子——一个钉子一个坑"

#### 实效吴昊 & 华与华方法论

**核心主张：超级符号 = 超级创意**

| 概念 | 解释 |
|------|------|
| 超级符号 | 人类熟悉的文化符号、图像、词语，无需解释直接传达 |
| 货架意识 | 产品即陈列，陈列即沟通，一切皆媒体 |
| 母体寄生 | 寄生在人类文化母体中的品牌符号 |

> "品牌就是符号，符号就是编码"
> "创意不是灵光一现，是解码母体"
> "产品是最大的媒体，包装是最大的广告"

代表案例：厨邦酱油绿格子、西贝莜面村"I love 莜"、得到猫头鹰

#### 小马宋

**核心主张：营销常识 + 创意克制**

> "营销不是做不可思议的事，是把理所当然的事做好"
> "好文案的标准只有一个：消费者看了会不会行动"
> "创意的前提是理解，不理解消费者就别说创意"

#### 东东枪

**核心主张：文案是手艺，要练**

> "文案不是灵感，文案是肌肉"
> "写不出来是因为读不够"
> "好文案都是改出来的"

### 通用文案创作流程

```
1. 洞察 → 2. 定位 → 3. 策略 → 4. 创意 → 5. 文案 → 6. 测试
```

| 步骤 | 关键问题 |
|------|----------|
| 洞察 | 消费者真正关心什么？有什么未被解决的冲突？ |
| 定位 | 品牌在消费者心中占据什么位置？ |
| 策略 | 用什么角度/框架来沟通？ |
| 创意 | 有没有戏剧性？能不能寄生到文化母体？ |
| 文案 | 说人话了吗？口语化了吗？ |
| 测试 | 目标用户看了会不会行动？ |

### 标题创作公式

| 公式 | 示例 |
|------|------|
| 数字 + 利益 | "3个技巧，让你多睡1小时" |
| 痛点 + 方案 | "便秘难受？试试这招" |
| 悬念 + 揭秘 | "月薪3千和3万的人，差的不只是钱" |
| 对比 + 反差 | "比咖啡更提神的，是这个" |
| 情绪 + 场景 | "为什么年轻人都不想结婚了" |
| 俗语 + 改造 | "有钱没钱，干净过年" |

### 卖点提炼方法

```
痛点 → 方案 → 利益 → 证明
```

### 优秀文案四感法则

| 标准 | 解释 |
|------|------|
| 力量感 | 简短有力，掷地有声 |
| 号召性 | 行动导向，驱动行为 |
| 文辞美 | 有节奏感、修辞、韵律 |
| 冲击力 | 直击人心、过目不忘 |
| 情感温度 | 自然流露的分享欲和情感共鸣 |

### 文案三大忌讳

| ❌ 忌讳 | ✅ 正确做法 |
|--------|------------|
| 空洞形容词 | 用具体场景替代（不说"品质生活"，说"树长大了、物业验证了"） |
| 说教感 | 让事实说话，不让品牌自夸 |
| 过长句式 | 读出来测试，超过三秒喘不过气就拆句 |

## 三、地产营销核心方法论

### 1. 定位三角

```
         我是谁（产品定位）
              △
             / \
            /   \
           /     \
          /       \
    我在哪  ──────  我卖给谁
   （区位定位）     （客群定位）
```

- **产品定位**：别墅/大平层/刚需首改/商业/文旅
- **客群定位**：年龄/职业/收入/家庭结构/价值观
- **区位定位**：核心区/新区/郊区/文旅板块

### 2. 价值梳理三层次

| 层次 | 内容 | 举例 |
|------|------|------|
| 基础价值 | 地段、户型、价格 | 地铁口、90㎡三房、2万/㎡ |
| 提升价值 | 品质、配套、物业 | 精装修、品牌物业、社区园林 |
| 差异价值 | 品牌、稀缺性、独特资源 | 知名开发商、一线江景、学区 |

**核心原则**：差异化价值 > 提升价值 > 基础价值

### 3. 地产常用定价策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 竞品对标法 | 与周边竞品价格对比定价 | 成熟片区 |
| 低开高走 | 首次开盘低价，后批次提价 | 首次开盘、渠道拓展期 |
| 高开低走 | 展示品牌形象，walk-in客户转化 | 高端项目、品牌开发商 |
| 价值定价法 | 基于产品力的溢价空间定价 | 稀缺产品、有独特卖点 |
| 心理价位测试 | 通过认筹/诚意登记摸底客户承受力 | 任何项目 |

### 4. 地产推广渠道矩阵

**线上渠道：**
- 抖音/快手：短视频+直播带看
- 小红书：种草笔记、生活方式内容
- 微信：公众号+私域社群+视频号
- 房产平台：贝壳/安居客/链家端口
- 搜索引擎：百度/360竞价

**线下渠道：**
- 案场体验：示范区/样板间/儿童区
- 经纪人渠道：二三级联动、经纪人大会
- 活动营销：节点活动/事件营销/圈层活动
- 派单/陌拜：传统地推
- 企业团购：政企大客户

**内容策略：**
- 价值故事化：站在业主角度讲生活场景
- 场景化呈现：不讲面积讲生活（"主卧大到能放婴儿床"）
- 信任状构建：开发商品牌/工程进度/已交付项目

### 5. 地产常用营销工具

| 工具 | 说明 |
|------|------|
| 首付分期 | 降低门槛，缓解客户资金压力 |
| 精装套餐 | 定制化装修方案，提升溢价 |
| 老带新奖励 | 佣金激励现有业主推荐 |
| 节日特惠 | 节假日限时折扣 |
| 一口价房源 | 特惠房源清货 |
| 团购优惠 | 批量采购谈判空间 |
| 认筹优惠 | 小额认筹锁定客户 |

### 6. 开盘前后关键动作

**蓄客期（开盘前1-3个月）：**
- 渠道铺排：经纪人/企业/圈层
- 认筹锁客：认筹金/诚意金
- 价格测试：VIP认筹摸底
- 示范区开放：体验式营销

**开盘期：**
- 集中签约：制造热销氛围
- 销控管理：好房源控场
- 优惠收紧：开盘后折扣回收

**续销期：**
- 持销推广：日常来访接待
- 分销激励：调整渠道佣金
- 特惠房源：清盘出货

## 四、地产文案创作特点

### 户型文案公式

| 维度 | 低端表达 | 高级表达 |
|------|---------|---------|
| 面积 | 90㎡ | 约90㎡三房 |
| 房间数 | 三室两厅 | 三代同堂，各有天地 |
| 采光 | 朝南 | 南向三开间，阳光全覆盖 |
| 收纳 | 有储物间 | 收纳空间等于多出一间房 |
| 主卧 | 主卧大 | 独立主卧套房，私密不打扰 |

### 区位文案公式

```
不只说"近"说"有多近"：
- 不说：距离地铁500米
- 说：地铁口500米，步行7分钟

不只说"配套"说"有什么配套"：
- 不说：周边配套成熟
- 说：出门三所幼儿园，隔壁三甲医院

用时间代替距离：
- 不说：距市中心10公里
- 说：开车25分钟到市中心
```

### 投资回报文案

| 要素 | 内容 |
|------|------|
| 租金回报 | 同地段租金对比，算账给客户 |
| 资产涨幅 | 区域规划利好，价格趋势 |
| 出租率 | 租赁市场活跃度 |
| 贷款计算 | 月供与租金对比 |

### 竞品打压话术

> **原则：不直接攻击竞品，而是用数据说话**

- 不说：竞品质量差
- 说："我们用的都是一线品牌，厨卫全是德系/日系"
- 不说：竞品位置偏
- 说："我们离地铁口最近，只有我们能做到X分钟步行"

## 六、快速检核清单

### 文案检核
- [ ] 货架测试：放在10条广告里能第一个被注意到吗？
- [ ] 说人话：读出来顺口吗？
- [ ] 有冲突/戏剧性吗？
- [ ] 可记忆：过耳不忘吗？
- [ ] 可行动：看完想不想行动？
- [ ] 可分享：愿意发给朋友吗？

### 营销策略检核
- [ ] 定位三角是否清晰（产品/客群/区位）
- [ ] 差异化卖点是否突出
- [ ] 定价是否有竞争力（与竞品对比）
- [ ] 渠道是否精准（渠道匹配客群）
- [ ] 营销节点是否卡好（蓄客/开盘/续销节奏）
- [ ] 案场动线是否顺畅
