# Somark Document Parser

## Overview

A document parsing skill for extracting content from various document formats.

## Supported Formats

- PDF documents
- Word documents (.docx)
- Images with text
- HTML files

## Usage

Use standard Python libraries like pypdf, python-docx, or docling for document parsing.
