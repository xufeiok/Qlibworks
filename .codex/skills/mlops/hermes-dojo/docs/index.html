The feedback loop
Measure weakness. Fix it. Measure again.

Monitor

Read session logs. Track tool errors, retries, user corrections.

Analyze

Rank failures by frequency and impact. Find root causes.

Fix

Patch failing skills. Create missing ones. Inject config.

Evolve

Run GEPA on weak skills. DSPy optimization with real data.

Report

CLI or Telegram. Deltas, sparklines, what changed.

Skill Creation
Generates new skills from failure patterns. Patches existing ones with targeted fixes.
GEPA Evolution
DSPy-based prompt optimization (ICLR 2026). Uses real failure data as the optimization signal.
Session Analysis
Reads past sessions from state.db. Identifies recurring failures, retry loops, user corrections.
Overnight Autopilot
Cron-scheduled improvement cycle. Runs while you sleep, reports in the morning.
Telegram Reports
Morning digests with deltas, sparklines, and what changed. Know before you open your laptop.
Learning Curve
Tracks success rate over time. Proves the feedback loop is working, not just running.
