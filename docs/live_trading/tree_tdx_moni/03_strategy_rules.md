# 策略执行规则

## 选股规则

- 使用 `score_tree.csv` 中最新交易日分数
- 先做行业约束
- 再按 `score >= 0.7` 过滤
- 最多保留 `TOP_K=20`
- 组合总权重为 `BUY_PCT=0.95`
- 首版采用等权分配

## 调仓规则

保持和回测基线一致：

- 周二为调仓信号日
- 周三为执行买入日

当前第一版执行器只负责将“当前持仓”调整为“目标持仓”。

## 执行规则

- 先卖出不在目标池中的持仓
- 再买入目标池中的缺口仓位
- 单票目标金额不超过总资产的 `MAX_SINGLE_ORDER_PCT`
- 下单数量按 100 股整手取整

## 代码规则

- 本地研究层统一使用 Qlib 格式：`600000.sh`
- 通达信执行层统一使用通达信格式：`600000.SH`
- 执行前做一次格式转换
