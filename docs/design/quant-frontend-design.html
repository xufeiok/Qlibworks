QlibWorks Quant Studio

一站式量化研究前端系统，融合 SuperPlotView TradingView 风格可视化与 quantdinger 现代因子管理范式。覆盖数据、因子、策略、回测、市场五大管理域。

React + Vite + TypeScript lightweight-charts Shadcn/ui + Tailwind FastAPI + WebSocket DuckDB + ClickHouse

09 核心数据流与 API

从前端到后端的完整链路，以 ML 因子策略全流程为例。

前端操作 > FastAPI > Celery > Python引擎 > WebSocket > 前端渲染
因子流程YAML编辑 > 预览 > 计算 > IC分析 > 合成 > 入库
训练流程数据加载 > 特征 > 切片 > Optuna > 训练 > 评估 > 注册
回测流程策略配置 > 数据准备 > BT运行 > 结果 > SuperPlot渲染
行情流程数据拉取 > 缓存 > K线 > 指标 > AI分析 > 推送

设计核心理念

融合 SuperPlotView 血统 - 沿用 TradingView 深色主题、lightweight-charts、SuperPlot 指标卡片

借鉴 quantdinger 范式 - 因子即配置、策略即组合、回测即分析

数据驱动决策 - 图表优先、表格补充、配置在侧

实时协作感 - WebSocket 实时推送

实施路线图

Phase 1: 脚手架+主题+路由+数据模块

Phase 2: 因子浏览器+IC分析+因子合成

Phase 3: 策略训练+Optuna+模型评估

Phase 4: 回测管理+SuperPlot+对比

Phase 5: 市场看板+K线+AI研报
